=== spring通用工具拓展

:basedir: commons

提供一些通用、工具类方便开发

该模块不强制绑定具体业务场景，适合在 starter、自定义自动配置和普通业务代码中复用。 +
其中 `annotation`、`aop`、`expression` 更偏扩展开发；`http`、`jackson`、`web`、`util` 更偏日常业务工具。

[primary]
.Maven
--
[source,xml,indent=0]
----
<dependency>
    <groupId>io.github.livk-cloud</groupId>
    <artifactId>spring-extension-commons</artifactId>
    <version>${version}</version>
</dependency>
----
--

[secondary]
.Gradle-Groovy
--
[source,groovy,indent=0]
----
    implementation 'io.github.livk-cloud:spring-extension-commons:${version}'
----
--

[secondary]
.Groovy-Kotlin
--
[source,kotlin,indent=0]
----
    implementation("io.github.livk-cloud:spring-extension-commons:${version}")
----
--
